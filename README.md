# [Easy Blogging Using Python and Pelican](https://github.com/dclong/blog)

## Help Doc 

./blog.py -h  

## Search for a Post 

./blog.py s pyspark pandas udf

## Edit a Post Returned in the SRPs 

./blog.py e 1 

## Add a new Post 

./blog.py a this is a new example post 

## Publish Sub Blogs
```
./blog.py p -ecmoF
```
