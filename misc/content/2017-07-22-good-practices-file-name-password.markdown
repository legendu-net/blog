## File Names

1. Avoid the following special characters in file names.
    - spaces (` `) 
    - dollar signs (`$`)
    - double quotes (`"`)
    - single quotes (`'`)
    - tilde (`~`)
    - dash/minus (`-`) 

2. When you programmally get the path of a file, 
    it is best to convert it to its absolute form.

## Password

1. Avoid the following special characters in passwords.
    - space (` `)
    - dollar signs (`$`) 
    - double quotes (`"`) 
    - single quotes (`'`)
